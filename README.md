# c5_c1_nuvexa_shop
<img width="830" height="517" alt="Captura desde 2026-03-10 07-20-02" src="https://github.com/user-attachments/assets/28b475d5-9ad6-4a6f-97f1-8e68740e08b1" />

This project automates the daily sales registration of a store.

The system allows the user to register multiple sales during the execution of the program. 
Each sale requires the product name, unit price, and quantity sold.

The program stores the information using a dictionary structure where the key represents 
the product name and the value represents the total quantity sold.

The system calculates the total revenue and generates a final summary showing the products 
sold, their quantities, and the total income of the day.
